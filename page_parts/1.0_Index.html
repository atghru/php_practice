Для заполнения формы обращения и регистрации нового пользователя, нажмите на кнопку "Заполнить форму обращения".

Для просмотра существующих обращений, нажмите на кнопку "Просмотр существующих обращений".

Внимание, для просмотра существующих обращений требуется авторизация (т.е. для пользователя должен быть выставлен флаг "Userlisting").
